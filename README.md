# Bëzic Manjakù

## Notre communauté, notre fierté

Application mobile communautaire pour rassembler la communauté manjak, favoriser les échanges, transmettre la langue et l'histoire, et connecter les associations.

---

## 📱 Fonctionnalités V1

### ✅ Implémentées

1. **Chat communautaire**
   - Salons de discussion thématiques
   - Messages en temps réel
   - Notifications de nouveaux messages

2. **Actualités & Événements**
   - Flux d'actualités communautaires
   - Calendrier d'événements
   - Détails complets avec participation

3. **Vidéos éducatives**
   - Bibliothèque de vidéos (Langue, Histoire, Culture, Témoignages)
   - Lecteur vidéo intégré
   - Recommandations de contenu

4. **Répertoire des associations**
   - Recherche par village, zone ou type
   - Fiches détaillées des associations
   - Coordonnées de contact

5. **Notifications**
   - Alertes pour messages, événements et actualités
   - Badge de notifications non lues

6. **Profil utilisateur**
   - Informations personnelles
   - Villages d'origine
   - Centres d'intérêt
   - Contenus sauvegardés

7. **Recherche globale**
   - Recherche unifiée dans tous les contenus
   - Résultats groupés par type

## 🎨 Design System

### Palette de couleurs

- **Vert profond** `#1E4D3A` - Primary
- **Terre cuite** `#C96A3D` - Accent
- **Sable** `#F4E9D8` - Light/Secondary
- **Ivoire** `#FFFDF8` - Background
- **Charbon** `#1F2937` - Text
- **Or doux** `#D4A64A` - Highlight

### Typographie

- Police : Sans serif moderne (système natif)
- Hiérarchie claire et lisible
- Espacements généreux

### UI Components

- Radius : 16px (1rem)
- Boutons : Primaire, Secondaire, Ghost
- Cards modernes avec ombres légères
- Icônes Lucide React
- États : Loading, Empty, Error, Success

## 🗺️ Structure de navigation

### Navigation principale (Bottom Tab Bar)

1. **Accueil** - Vue d'ensemble et accès rapides
2. **Discussions** - Salons de chat communautaires
3. **Actualités** - News et événements
4. **Vidéos** - Bibliothèque éducative
5. **Associations** - Annuaire des associations

### Navigation secondaire

- **Recherche** - Accessible depuis l'accueil
- **Notifications** - Accessible depuis l'accueil
- **Profil** - Accessible depuis les paramètres
- **Paramètres** - Accessible depuis le profil

## 🚀 Développement

### Installation

```bash
npm install
```

### Démarrage

```bash
npm run dev
```

L'application sera accessible sur `http://localhost:5173`

### Build

```bash
npm run build
```
